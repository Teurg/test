# Использование 

Проверяем создание базы

```
docker exec -it a088f023b53d bash
```

Заходим под польозователем postgres

```
psql -U postgres
```

Смотрим наличие баз

```
\l
```

Проверяем список ролей

```
\du
```

Можно подключиться к бд для проверки

```
\c grafana;
```

И посмотреть таблицы

```
\dt
```
